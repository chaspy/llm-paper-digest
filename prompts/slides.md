# Marp スライド生成プロンプト

summaries/ の日本語解説をもとに Marp スライドを生成する。

## スライド構成（7-10枚）

1. タイトルスライド（論文名・著者・年・一行サマリー）
2. 背景と問題
3. 核心アイデア（図または箇条書き）
4. 手法の詳細
5. 実験設定
6. 結果と考察
7. 影響と応用
8. まとめ

## Marp フロントマター
```yaml
---
marp: true
theme: default
paginate: true
---
```

## 制約
- 1スライド 3-5 行以内
- コードブロックは最小限
- 日本語のみ
