# llm-paper-digest

LLM・AI 論文の日本語解説と Marp スライドを自動生成するリポジトリ。

## 構成

- `lists/reading-list.md` — 対象論文リスト
- `papers/` — 論文メタデータ
- `summaries/` — 日本語詳細解説
- `slides/` — Marp スライド（HTML 変換後 GitHub Pages で公開）

## GitHub Pages

スライドは GitHub Pages で公開: https://chaspy.github.io/llm-paper-digest/

## 使い方

Worker（Claude Code）が `lists/reading-list.md` の pending 論文を順に処理する。
